## XIA Project

🥝 XIA Project 是一个个人项目，纯粹是为了满足个人日常使用，
该系统仅包含了基本的框架功能，如：用户管理，菜单管理，权限管理，方便在该基础上搭建出各种系统。

该系统仍在持续开发中，期望可见变成一个强大的系统，
如果你有兴趣并想进一步了解，欢迎。

快速上手指南：[doc/QUICK_START.md](doc/QUICK_START.md)

#### 技术栈：

- MySQL 5.7
- Java 11
- Spring Boot 2.1
- Spring Data JPA
- Spring Security
- EasyUI over jQuery

系统架构清晰明了，复杂度堪比 Hello World ！
